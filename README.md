2048 Clone

A modern implementation of the classic 2048 game, built with React, TypeScript, and Vite.

This project was developed as part of a interview process. 

🎮 Demo
Play the game here: https://wilsonchng.github.io/play2048/ 

🚀 Tech Stack
React – UI framework
TypeScript – Type safety and scalability
Vite – Fast build tool and dev server
Plain vanilla HTML & CSS  (no fancy libraries)

✨ Features
Fully playable 2048 game
Keyboard controls (arrow keys)
Tile merging logic
Score tracking and persistance to browser (localStorage)
Game over detection
Restart game functionality
